# Introdução

Este bot abre o executável da calculadora e realizar determinado calculo selecionado pelo usuário!

## Bibliotecas utilizadas

- sys (nativo)
- os  (nativo)
- pywinauto (externo)
- pywin32 (externo)
- subprocess (nativo)
- time (nativo)
- Logger (nativo)
