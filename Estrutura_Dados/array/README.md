# O que é um array? 

É uma coleção indexada de dados. Indexado significa que um array armazena uma sequência de itens (geralmente chamados de elementos) e só podemos acessar cada elemento pela sua posição (conhecido como índices). 

## Garacterísticas Gerais

- Armazena uma coleção de dados;
- Seus elementos podem ser acessados pelo índicie;
- Os elementos não precisam ser acessados de modo sequencial:
    - Isto é: posso acessar qual quer posição se ler sua posição antecessora;
    - exemplo: acessar elemento de índicie 10, sem precissar passar pelo elemento de índice 9.

## Garacterísticas Específicas

- Arrays são alocados na memória como um bloco initerrupto com locais sequencias, o que é eficiente tanto para o uso da memória quanto para a rapidez;

- Os arrays estão restritos ao armazenamento de dados do mesmo tipo (homogêneos).Essa restrição também vem da necessidade de otimização, porque permite que mesma memória seja alocada para cada elemento do array e o copilador/interpretador saiba rapidamente o endereço de memória de cada elemento.

- O numero de elementos contidos no array, deve ser definido quando for criado e esse tamanho não pode alterado
